# Vrinda Store Sales Analysis – Excel Dashboard

This project analyzes the sales data of **Vrinda Store** using Microsoft Excel.  
The goal is to explore customer behavior, identify sales trends, and generate business insights through data analysis and visualization.

## Project Objective

The objective of this project is to analyze the annual sales data of Vrinda Store and create an **interactive Excel dashboard** that helps understand customer purchasing patterns and improve business decisions.

## Dataset Overview

The dataset includes information such as:

- Order ID
- Customer gender and age
- Product category
- Sales amount
- Order status
- Sales channel
- Shipping state
- Order date

This data is used to analyze trends and identify key factors that influence sales performance.

## Analysis Performed

The following analysis was performed using Excel:

- Sales vs Orders comparison
- Monthly sales trend analysis
- Orders by gender
- Orders by age group
- Top performing states
- Order status distribution
- Sales channel contribution
- Best selling product categories

## Tools & Techniques Used

- Microsoft Excel
- Data Cleaning
- Pivot Tables
- Pivot Charts
- Slicers
- Data Visualization

## Key Insights

Some important insights from the analysis:

- Women contribute the majority of orders (~65%)
- Adult age group (30–49 years) contributes the most sales
- Online channels like Amazon, Flipkart, and Myntra generate most orders
- Maharashtra, Karnataka, and Uttar Pradesh are top contributing states
- Certain product categories drive the majority of revenue
